// Capabilities

Features

🔍

Real-Time Detection

YOLOv8 nano model runs inference on every frame — detecting backpacks, handbags, and suitcases with confidence scoring.

🧠

Centroid Tracking

Each object is assigned a unique ID and tracked across frames using nearest-centroid matching. New objects register automatically.

⏱️

Per-Object Timer

Every tracked object has its own stationary timer. Movement resets the clock. Stillness counts down to the alert threshold.

🚨

Visual Warnings

Unattended objects receive a red bounding box and "UNATTENDED!" label. A full-width warning banner appears at the top of the frame.

🎛️

Configurable Thresholds

Adjust the alert timer, movement tolerance, and confidence threshold via simple constants at the top of main.py.

📹

Webcam or Video File

Works with a live webcam feed or any pre-recorded video file — just change one line in the source configuration.

// Architecture

How It Works

1 · Video Frame Capture

OpenCV reads each frame from the webcam or video file at native FPS.

2 · YOLOv8 Inference

The frame is passed to YOLOv8. Results are filtered to backpack / handbag / suitcase with confidence ≥ 0.4.

3 · Centroid Extraction

Each bounding box center is computed and fed to the ObjectTracker.

4 · Track Matching

New centroids are matched to existing tracks by Euclidean distance. Unmatched detections spawn new tracks.

5 · Stationary Timer

If an object moves > 30px → reset timer. If it stays still → timer accumulates. At 5 seconds → UNATTENDED.

6 · Annotated Output

Frame is drawn with bounding boxes, labels, timers, and warning overlays, then displayed via cv2.imshow().
